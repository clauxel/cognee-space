Site policy

Privacy

Cognee Space stores only the operational events needed to run the planner, pricing flow, analytics, and support.

Input handling

The public planner is for deployment planning. Do not enter secrets, API keys, regulated records, or private production memory into support messages.

Analytics

Cloudflare D1 stores page views, CTA clicks, pricing toggles, checkout starts, paid gate hits, planner events, and referral signals.

Support

Support requests go to the listed contact address and should include receipts or non-sensitive workflow details only.

Source boundary

Official project facts are cited as evidence in source notes. Conversion and checkout paths remain on this domain.
